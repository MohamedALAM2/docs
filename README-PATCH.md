# CrimeLens diagram patch

Copy these files into the repository root and commit them:

- `index.html`
- `assets/diagram-enhancements.css`
- `assets/diagram-enhancements.js`

This patch replaces the ASCII/code-block diagrams with responsive visual diagrams for:

1. `Overview > The Approach: AI-Originated CAD`
2. `Architecture > High-Level Diagram`

It is frontend-only and works with the existing static SPA. No backend or build step is required.
